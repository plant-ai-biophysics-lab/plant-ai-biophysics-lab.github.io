# UC Davis Plant AI Biophysics Lab — Website

This repository is the source for the [Plant AI and Biophysics Lab (PAIBL)](https://plant-ai-biophysics-lab.github.io) website, hosted on GitHub Pages.

## Run locally

1. Install Ruby and Bundler, then install dependencies:

   ```bash
   bundle install
   ```

2. Serve the site (default: http://localhost:4000):

   ```bash
   bundle exec jekyll serve
   ```

3. Open [http://localhost:4000](http://localhost:4000) in your browser.

## Tech

- **Jekyll** with the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) theme (remote theme).
- Content: `_pages/` (Blog, Projects, Team, Papers, Code); team data: `_data/team.yml`.
